# 角色属性显示插件 - 设计文档

## 版本 2.4.0

---

## 一、功能概述

本插件是一个魔兽世界界面增强插件，用于在游戏界面上实时显示角色的关键属性信息，包括装等、主属性、绿字属性和移速等。

---

## 二、界面设计

### 2.1 主界面布局

```
┌────────────────────────┐
│ 角色属性               │  ← 标题左对齐
├────────────────────────┤
│ 装等               226.8│

│ 暴击               17.7%│
```

### 2.2 框体规格

| 属性 | 值 | 说明 |
|------|-----|------|
| 宽度 | 170px | 固定宽度 |
| 高度 | 动态计算 | 根据显示属性数量自动调整 |
| 左右边距 | 12px | 标签和值两侧留白 |
| 行高 | 跟随字体 | 由字体大小和固定间距共同决定 |
| 标题高度 | 跟随字体 | 比正文大 2px |
| 数值列 | 60px | 固定右对齐列宽 |

## 三、属性显示顺序

按照魔兽世界默认的角色面板顺序排列：

1. **装等** - 角色平均装备等级
2. **主属性** - 力量/敏捷/智力（根据职业自动识别）
3. **暴击** - 暴击几率
4. **急速** - 急速百分比
5. **精通** - 精通效果
6. **全能** - 全能百分比
7. **吸血** - 生命偷取（可选，默认隐藏）
8. **招架** - 招架几率（可选，默认隐藏）
9. **闪避** - 闪避几率（可选，默认隐藏）
10. **格挡** - 格挡几率（可选，默认隐藏）
11. **移速** - 移动速度百分比

| 职业 | 主属性 | 说明 |
|------|--------|------|

| 圣骑士 | 力量/智力 | 神圣=智力，其余=力量 |
| 死亡骑士 | 力量 | - |
| 猎人 | 敏捷 | - |
| 潜行者 | 敏捷 | - |
| 恶魔猎手 | 敏捷 | - |
| 武僧 | 敏捷/智力 | 织雾=智力，其余=敏捷 |
| 法师 | 智力 | - |
| 牧师 | 智力 | - |
| 术士 | 智力 | - |
| 德鲁伊 | 智力/敏捷 | 平衡/恢复=智力，野性/守护=敏捷 |

| 唤魔师 | 智力 | - |

## 五、设置界面设计

### 5.1 打开方式

- 聊天命令：`/csd` 或 `/csd config` 或 `/csd settings`

### 5.2 界面布局

```
┌──────────────────────────────────────┐
│ 角色属性显示 - 设置            [_][X]│
├──────────────────────────────────────┤
│                                      │
│ 框体移动                             │
│ ┌────────────┐                       │
│ │ 锁定框体   │  ← 点击切换锁定状态   │
│ └────────────┘                       │
│                                      │
│ 外观                                 │
│ 字体大小      14   [-] [+]           │
│ 背景透明度    0.8  [-] [+]           │
│ 面板样式   无边框   [下拉选择]       │
│ 行距          6    [-] [+]           │
│                                      │
│ 额外属性显示                         │
│ ☑ 显示吸血                          │
│ ☐ 显示招架                          │
│ ☐ 显示闪避                          │
│ ☐ 显示格挡                          │
│                                      │
│ ┌────────────┐      ┌────────┐       │
│ │ 重置位置   │      │  关闭  │       │
│ └────────────┘      └────────┘       │
│                                      │
└──────────────────────────────────────┘
```

### 5.3 功能说明

| 功能 | 说明 |
|------|------|
| 锁定/解锁框体 | 切换框体的可拖动状态 |
| 字体大小 | 调整主面板标题和属性文本大小 |
| 背景透明度 | 调整主面板背景透明程度 |
| 面板样式 | 在无边框、默认边框之间切换 |
| 行距 | 调整属性行与行之间的间距 |
| 显示吸血 | 勾选后在主界面显示吸血属性 |
| 显示招架 | 勾选后在主界面显示招架属性 |
| 显示闪避 | 勾选后在主界面显示闪避属性 |
| 显示格挡 | 勾选后在主界面显示格挡属性 |
| 重置位置 | 将框体重置到屏幕左下角默认位置 |
| 关闭 | 关闭设置窗口 |

## 六、数据存储

### 6.1 存储变量

使用 `CharacterStatsDisplayDB` 存储用户配置：

```lua
{
    point = "BOTTOMLEFT",           -- 锚点位置
    relativePoint = "BOTTOMLEFT",   -- 相对锚点
    xOfs = 20,                      -- X偏移
    yOfs = 20,                      -- Y偏移
    locked = false,                 -- 是否锁定
    panelStyle = "minimal",         -- 面板样式
    fontSize = 14,                  -- 字体大小
    bgAlpha = 0.8,                  -- 背景透明度
    lineSpacing = 6,                -- 行距
    showLeech = false,              -- 显示吸血
    showParry = false,              -- 显示招架
    showDodge = false,              -- 显示闪避
    showBlock = false,              -- 显示格挡
}
```

### 6.2 存储说明

- 配置自动保存，无需手动操作
- 位置信息在拖动框体后自动保存
- 设置更改后立即生效

## 七、更新机制

### 7.1 更新频率

| 属性类型 | 非战斗状态 | 战斗状态 | 说明 |
|----------|------------|----------|------|
| 装等 | 3秒 | 3秒 | 变化频率低 |
| 主属性 | 3秒 | 0.5秒 | buff可能影响 |
| 绿字属性 | 3秒 | 0.5秒 | buff频繁触发 |
| 移速 | 0.5秒 | 0.5秒 | 实时变化 |

### 7.2 优化策略

1. **脏检查**：属性值未变化时不重绘UI
2. **战斗感知**：自动检测战斗状态调整更新频率
3. **事件驱动**：装备更换、专精切换时立即更新

## 九、技术实现

### 9.1 核心API使用

| 功能 | API |
|------|-----|
| 装等 | `GetAverageItemLevel()` |
| 力量 | `UnitStat("player", 1)` |
| 敏捷 | `UnitStat("player", 2)` |
| 智力 | `UnitStat("player", 4)` |
| 暴击 | `GetCritChance()` |
| 急速 | `GetHaste()` |
| 精通 | `GetMasteryEffect()` |
| 全能 | `GetCombatRatingBonus(CR_VERSATILITY_DAMAGE_DONE)` |
| 吸血 | `GetLifesteal()` |
| 招架 | `GetParryChance()` |
| 闪避 | `GetDodgeChance()` |
| 格挡 | `GetBlockChance()` |
| 移速 | `GetUnitSpeed("player")` |

### 9.2 事件监听

- `PLAYER_LOGIN` - 登录初始化
- `PLAYER_EQUIPMENT_CHANGED` - 装备更换
- `PLAYER_REGEN_DISABLED` - 进入战斗
- `PLAYER_REGEN_ENABLED` - 脱离战斗
- `ACTIVE_TALENT_GROUP_CHANGED` - 天赋切换
- `PLAYER_SPECIALIZATION_CHANGED` - 专精切换
